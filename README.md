# dynamic-frontend-showcase

Moderní onepage demo s těmito částmi:

- fotogalerie s otevřením v novém okně a automatickou slideshow po 1 s,
- týdenní kalendář s navigací mezi týdny, zvýrazněním víkendů a dneška,
- jednoduché piškvorky s alternací tahů,
- EuroJackpot losování 5 z 50 a 2 z 11,
- registrační formulář s průběžnou validací,
- generátor sudoku.

Otevřete soubor [index.html](index.html) v prohlížeči.
